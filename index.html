Quick Notes

Simple operating rules for keeping this repository clean and expandable.

Folder Pattern

Keep each topic inside its own folder and use a consistent page name such as pre-analysis.html for easier linking and expansion later.

Future Main Dashboard

This page can later evolve into a full dashboard with categories, filters, company-wise sections, and updated discussion status blocks.

Best Naming Style

Use lowercase folder names with underscores or hyphens only. Avoid spaces so the URLs stay clean and predictable.
